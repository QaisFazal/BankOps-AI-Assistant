# Assumptions

- The first milestone is a clean project scaffold, not a production assistant.
- Real enterprise authentication is out of scope for the initial placeholder.
- Pinecone, LangSmith, and model credentials will be supplied through environment variables.
- Sample documents are safe mock data and should not include customer secrets.
- LangGraph nodes will be added after the assistant's required workflows are agreed.
- Streamlit is used for demo speed rather than final enterprise UI polish.
